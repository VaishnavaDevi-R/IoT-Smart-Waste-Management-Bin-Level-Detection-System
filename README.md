# IoT Smart Waste Management & Predictive Bin Monitoring System

An IoT-based Smart City solution for monitoring waste bins using ESP32, HC-SR04, MQTT, Node-RED, ThingSpeak, and Streamlit dashboards.

## Features

- Real-Time Bin Monitoring
- Fill Percentage Detection
- MQTT Communication
- Node-RED Dashboard
- ThingSpeak Cloud Integration
- Streamlit Analytics
- Alert Generation
- Predictive Overflow Analysis
